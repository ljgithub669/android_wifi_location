## 一、定位的需求[背景](https://blog.csdn.net/huawei_eSDK/article/details/50777339)

- 工厂管理者希望精确追踪贵重物资和生产设备
- 消防员在滚股浓烟中需要进行有效救援
- 商场中，消费者希望准确定位商铺和货品
- 销售者希望实现精准的广告推送
- 智能家居，需要精准定位使用者位置
- 医院，价值价值千百万的贵重医疗设备需要管控
- 大人需要知晓人群中孩子的位置

- 解决方案的后期维护成本太高
- 产业链不成熟导致成本较高
